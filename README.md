## Run Locally

Clone the project

```bash
  git clone https://github.com/Mohit-Thapliyal/Expenses-react-app.git
```

Go to the project directory

```bash
  cd Expenses-react-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
